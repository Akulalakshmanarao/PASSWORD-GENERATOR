

# Password Generator

## Project Overview

The Password Generator is a simple project that creates strong and secure passwords based on user-defined preferences. It helps users generate random passwords to improve account security and reduce the risk of weak or repeated passwords.

## Features

* Generates random and secure passwords
* Supports different password lengths
* Option to include uppercase letters, lowercase letters, numbers, and special characters
* Simple and beginner-friendly project
* Useful for learning Python logic and string handling

## Tech Stack

* Python 3.x
* Built-in Python libraries (no external dependencies required)

## How to Run

1. Install Python on your system
2. Download or clone the project files
3. Run the program using your system terminal or Python editor

## Usage

1. Start the program
2. Choose the desired password length
3. Select the type of characters to include
4. The program will generate and display a secure password
